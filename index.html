Python Tutorial

 Hey! Python is one of the most used coding languages. It is used for many things such as: Game development, Machine learning and Automating tasks.

 To continue, you will need to download python which can be found at https://www.python.org/downloads/
 Make sure to click "Add to PATH" during installation

 The first thing most people learn in any coding language is how to output "Hello World!" into the console. In Python the print() command is used. Before we go any further let's talk
 about variables and their types.

 A variable is a value that can change, depending on conditions or information passed to the program. Each variable type is used for a specific function. This will be explored in-depth later
 The types are:
   - Strings: A "string" of words eg. "The quick brown fox jumps over the lazy dog"
   - Integers (Int): Whole numbers eg. 8
   - Float: Decimals eg. 3.14159
   - Boolean (Bool): A boolean is True/False eg. True
   - Character (Char): A single character eg. 'a'
   - List: A list of strings, ints and floats eg. ["hello", "world", 0]
   - Dictionary: Essentially a list of a pair of two values; A key and a value. If the program requests a key's value it will return it's matching value eg. {"apple": "a red fruit", "banana": "a
     yellow fruit"}

 A parameter/s can be passed to a function to do something with that parameter. You always put the parameter inside brackets.
 The print() function takes a parameter and outputs it in the console. When working with strings in python always place them in a "". To print something write print("your string here")
 Now you try it!

 print("Hello World!")
 Output:
 Hello World!
